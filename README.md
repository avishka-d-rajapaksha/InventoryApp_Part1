# Inventory Manager - Part 1 (TypeScript Web App)

A **professional Inventory Management Dashboard** built with TypeScript and vanilla HTML/CSS. Part 1 of the PROG2005 Programming Mobile Systems assignment, featuring a modern semantic HTML structure with full CRUD operations for inventory items.

## Key Features

- **Add Inventory Items** — Create items with ID, name, category, quantity, price, supplier, and stock status
- **Delete Items** — Remove items by name or ID with confirmation
- **Search Items** — Find items quickly by name or ID
- **Update Items** — Modify specific fields of existing items
- **Item Preview** — Live preview card before adding items
- **Auto-Suggestions** — Smart suggestions for names, categories, and suppliers
- **Popular Items** — Mark and filter popular inventory items
- **Responsive Dashboard** — Professional sidebar + content area layout

## Technologies & Tech Stack

| Technology | Purpose |
|---|---|
| TypeScript | Type-safe application logic |
| HTML5 | Semantic page structure |
| CSS3 | Professional dashboard styling |
| JavaScript (ES6) | Compiled TypeScript output |
| LocalStorage | Client-side data persistence |

## Setup and Installation

1. Clone the repository:
   ```
   git clone https://github.com/avishka-d-rajapaksha/InventoryApp_Part1.git
   ```
2. If modifying TypeScript, compile with:
   ```
   tsc
   ```
3. Open `index.html` in a modern web browser, or use `Server.bat` to start the local server.

## Usage

- Fill in the **Add New Item** form and click **+ Add Item**
- Use **Display All Items** to view the full inventory table
- Search, update, or delete items using the respective sidebar forms
